# Nanny App

A Python/Flask web app for nanny/nursery/nounou/assistante maternelle.

## Features

* Manage children
* Manage parents
* Activity timeline (arrival, departure, food, bottle, activity)
* Family form with custom fields (Birth story, alergen...)

## Contributing

If you use vscode, open the repo in Devcontainer mode, run task `Install all requirements` and run F5 to run the flask server, then go to http://127.0.0.1:5000.
